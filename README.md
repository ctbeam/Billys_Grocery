# Billy's Admin Grocery Portal

BAGP is an application that gives users of a store the ability to create employee's as well as add inventory to the store database.

## Installation


```bash
TO RUN LOCALLY
npm install backend
npm install frontend
npm start backend
npm start frontend 
```
